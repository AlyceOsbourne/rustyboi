RustyBoi Web Emulator

A Game Boy emulator built for programmatic control and AI research

About RustyBoi

RustyBoi was created to provide a better platform for testing neural networks, such as NEAT (NeuroEvolution of Augmenting Topologies) and other AI algorithms. The core emulator is designed with programmatic use in mind, making it ideal for research, experimentation, and automated gameplay. This web interface provides a visual way to interact with the emulator, but the underlying engine is built to be controlled programmatically, enabling researchers and developers to easily integrate Game Boy emulation into their AI training pipelines.
