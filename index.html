RustyBoi Web Emulator

RustyBoi is a Game Boy emulator written in Rust, designed with programmatic use in mind. The core emulator is built to provide a robust platform for testing neural networks, including evolutionary algorithms like NEAT (NeuroEvolution of Augmenting Topologies). Whether you're training AI agents to play games, researching reinforcement learning, or building automated testing frameworks, RustyBoi offers a clean, performant API for programmatic control and integration.
